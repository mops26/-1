## Тестовые функции

Для анализа выбраны 3 функции с разными характеристиками:

| № | Функция | Отрезок | Точное значение | Особенности |
|---|---------|---------|-----------------|-------------|
| **f₁** | $x^5 - 5x^3 + 4x + 1$ | $[0, 2]$ | $\displaystyle \frac{2^6}{6} - \frac{5\cdot2^4}{4} + \frac{4\cdot2^2}{2} + 2 \approx 2.6667$ | Полином 5-й степени |
| **f₂** | $\dfrac{1}{\sin^2 x} = \csc^2 x$ | $[0.5, 2]$ | $\cot(0.5) - \cot(2) \approx 1.0780$ | Уходит в $\infty$ при $x \to 0$ |
| **f₃** | $e^{-x^2}$ | $[0, 3]$ | $\displaystyle \frac{\sqrt{\pi}}{2} \text{erf}(3) \approx 0.8862$ | Интеграл ошибок (неберущийся) |
---
**test_functions.py** содержит тестовые функции и их точные значения. Каждая функция имеет пояснение. Точные значения считаются аналитически: для полинома через первообразную, для sin⁻²x через котангенс, для ступеньки геометрически. Для e⁻ˣ² используется табличное значение, так как это интеграл ошибок.

**quadrature.py** реализует четыре квадратурные формулы. Все формулы составные, работают для любого количества отрезков. Для методов Симпсона и трех восьмых есть автоматическая коррекция n, если n оказывается нечетным числом

**analysis.py** строит графики
